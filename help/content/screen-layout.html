Screen Layout and Functions

Map Layout

Fig 2.1 - System Mapper Screenshot

The Screen is laid out into functional areas, most are self-explanatory, but some may need a little explanation. One area in particular that has multiple uses, is the Top-Middle Textarea. This area provides a place where the user can paste both D-Scan for treat analysis(Ships etc), D-Scan for Trilateration (Celestials) and Probe Scans. The buttons that initiate teir functionality is:

  • Analyse Current D-Scan - This button will take the ships from textarea and calculate and provide a D-Scan Analysis.
  • Parse D-Scan and Select Refs - This button will take the D-Scan data from the textarea and calculate your location in space.
  • Probe Scan - This button will take the Probe Scan data from the textarea and display it in the Probe Scan Results.
Prior to pasting new scans into the text area, use the 'Clear Scan Text' button to clear the previous data.

Map Area

  • Header Area
    Displays Current System, System Class, System Type Statics. To the right of this, are some system related controls: Select System text Box, Load System Button, Save and Share State Button and Help Pages Button
  • Map Area
    Displays the current map of the system, Kill Locations (+),Custom Markers, Range Rings and 4AU Celestial Spawn areas
  • Scan Paste TextArea
    This where the user can paste their D-SCan and Probe Scans for further analysis
  • Button And Control Araea
    Buttons for map and scan functionality
  • Plotted LocationsDisplays plotted locations from Trilateration and Custom mmarkers. Items here can be linked to Probe Scans
  • Recent Kill Activity
    Displays a list of kills in system during the last 12 hours(to a max of 10 kills)
  • D-Scan Threat Analysis
    When activated displays a breakdown of ships scanned in system
  • Add Custom Markers
    Provides the user with an interface to add custom markers to the map
The Map is a SVG area that displays the system and and user added content
